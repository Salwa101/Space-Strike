# Space-Strike


<h1>🚀 Space Strike</h1>

<p>
<strong>Space Strike</strong> is a simple 2D space shooting game made using <strong>Visual C++</strong> and the <strong>iGraphics</strong> library.  
In this game, you control a spaceship, shoot enemies, and try to survive as long as possible.
</p>

<h2>📄 About the Project</h2>

<p>
This project is made for learning basic game development using Visual C++.  
It includes movement, shooting, enemy spawning, collision detection, and score tracking.
</p>

<h2>🚀 How to Run</h2>

<ol>
  <li>Download or clone the project:
    <pre><code>git clone (https://github.com/Salwa101/Space-Strike)</code></pre>
  </li>

  <li>Open the project in Visual Studio</li>

  <li>Open <code>Space_Strike.sln</code></li>

  <li>Make sure <strong>iGraphics</strong> is set up properly</li>

  <li>Build and run the project</li>
</ol>

<h2>🛠️ Technologies Used</h2>

<ul>
  <li>Visual C++</li>
  <li>iGraphics Library</li>
</ul>

<h2>🎮 Features</h2>

<ul>
  <li>Move spaceship using keyboard</li>
  <li>Shoot enemies</li>
  <li>Enemy spawning system</li>
  <li>Collision detection</li>
  <li>Score system</li>
</ul>
